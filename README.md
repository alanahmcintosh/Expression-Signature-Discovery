# OncoSigBench

OncoSigBench is a benchmarking framework for evaluating methods that derive gene expression signatures associated with genetic alterations (mutations, CNAs, fusions). 

This project provides:
- A simulation engine for generating mutation, CNA, and expression data with ground-truth signatures.
- Multiple supervised and unsupervised methods for signature discovery.
- Evaluation metrics (accuracy, precision, recall, F1, MCC, etc.) to compare predicted signatures against ground truth.
- Summaries and visualizations across datasets.
